# ШІ-Асистент для аналізу зображень (Telegram Bot)

## Опис проєкту
Цей проєкт реалізує Telegram-бота, який використовує ШІ для аналізу зображень. Бот виступає в ролі технічного експерта, який детально описує об'єкти на надісланих фотографіях. Користувачі можуть додавати текстові підписи до фотографій, щоб ставити конкретні запитання щодо зображення, або відправляти фото без підпису для отримання загального опису.

Проєкт створено для демонстрації підходу AI-assisted coding та інтеграції зовнішніх API у .NET застосунок.

## Технологічний стек та залежності
- **Платформа:** C# / .NET 8 (або новіше)
- **NuGet-пакети:**
  - `Telegram.Bot` — для взаємодії з Telegram Bot API (отримання повідомлень та завантаження фото).
  - `OpenAI` — офіційний клієнт для звернення до моделей OpenAI (використовується Responses API).
  - `Microsoft.Extensions.Hosting` — для керування життєвим циклом застосунку, Dependency Injection та конфігурації.

## Структура проєкту
- `TelegramBotService.cs` — фоновий сервіс (реалізує `IHostedService`), який відповідає за прослуховування нових повідомлень з Telegram та базову маршрутизацію (обробка команд, перевірка типів повідомлень).
- `FrameProcessor.cs` — сервіс бізнес-логіки. Формує промпти, додає до них системні інструкції та зображення, відправляє запит до OpenAI API та повертає результат.

## Доступні команди
Бот підтримує такі базові команди:
- `/start` — привітання та короткий опис призначення бота.
- `/help` — інструкція з використання (як правильно надсилати фото та текст).

## Інструкція із запуску

### 1. Передумови
- Встановлений локально [.NET SDK](https://dotnet.microsoft.com/).
- Створений бот у Telegram (через [@BotFather](https://t.me/BotFather)) та отриманий **Bot Token**.
- Акаунт OpenAI та отриманий **API Key**.

### 2. Конфігурація середовища
Створіть файл `appsettings.json` у корені проєкту (поряд із `.csproj`) та додайте туди свої ключі доступу. 

> **Важливо:** Цей файл додано до `.gitignore`, щоб уникнути витоку секретних ключів у публічний репозиторій.

Формат файлу `appsettings.json`:
```json
{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.Hosting.Lifetime": "Information"
    }
  },
  "BotConfiguration": {
    "BotToken": "ВАШ_TELEGRAM_ТОКЕН_ТУТ"
  },
  "OpenAI": {
    "ApiKey": "ВАШ_OPENAI_КЛЮЧ_ТУТ",
    "Model": "gpt-4o-mini",
    "SystemPrompt": "Ти — експерт-аналітик зображень. Твоя мета — детально, але структуровано описувати те, що ти бачиш на надісланих користувачем фотографіях чи скріншотах. Правила роботи: 1. Спочатку дай короткий загальний опис зображення (1-2 речення). 2. Далі виділи ключові деталі, об'єкти або текст. 3. Якщо це графік чи схема — поясни їхню суть. 4. Відповідай українською мовою, чітко і без води."
  }
